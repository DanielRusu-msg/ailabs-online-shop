# Online Shop UI

Angular-based frontend for the online shop application. Provides a modern e-commerce interface with product browsing, shopping cart, and order management.

**Requirements:** Node.js >= 24.0.0

## Bootstrap

- Install Angular CLI: `npm install -g @angular/cli`
- Install dependencies: `npm install`
- Start development server: `npm start` (available at `http://localhost:4200/`)
- Start with mock backend: `npm run start:mock`
- Run tests: `npm test`
- Build for production: `npm run build`
