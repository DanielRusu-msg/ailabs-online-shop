# Online Shop

A small-scale full-stack e-commerce application with an Angular frontend and Spring Boot backend.

## Documentation

- [Project Overview](./docs/PROJECT.md) — user roles, features, and application scope
- [Backlog](./docs/BACKLOG.md) — remaining work items by stack and complexity

## Setup

- Clone the repository
- Create your own personal repository and clone it locally
- Copy **only** the folder which contains the app (`fullstack-onlineshop/`)
- Push the changes
- Start your [database](./docker/development/docker-compose.yml)
- Bootstrap your [Backend](./onlineshopapi/README.md)
- Bootstrap your [UI](./onlineshopui/README.md)
- Test the application is working [locally](http://localhost:4200)
